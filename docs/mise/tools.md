# Управление инструментами

## Блок [tools]

Все инструменты проекта перечисляются в `mise.toml` в блоке `[tools]`:

```toml
[tools]
bun   = "1.2.0"
node  = "22.0.0"
fnox  = "1.25.1"
age   = "1.3.1"
```

Версии фиксированы — все в команде работают с одним и тем же окружением.

## Добавить инструмент

```bash
mise use bun@1.2.0
```

Это добавит строку в `mise.toml` и сразу установит инструмент.

## Обновить версию

Просто меняешь версию в `mise.toml` и запускаешь:

```bash
mise install
```

## Посмотреть доступные версии

```bash
mise ls-remote bun
```

## Как это работает под капотом

Mise хранит инструменты в `~/.local/share/mise/installs/`. Когда ты заходишь в папку с `mise.toml` — shell hook подменяет `$PATH` так, чтобы нужные версии шли первыми. Ничего не устанавливается глобально в систему.
