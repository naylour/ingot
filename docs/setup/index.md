# Настройка проекта

Здесь описан полный процесс первичной настройки — от клонирования репозитория до работы с секретами.

## Что понадобится

- Git
- curl (для установки mise)
- Доступ к репозиторию

Всё остальное — Bun, fnox, age и другие инструменты — устанавливает mise автоматически.

## Порядок действий

1. [Установка mise и клонирование проекта](./first-run.md)
2. [Генерация ключа и получение доступа к секретам](./secrets-access.md)

Если ты добавляешь нового участника в команду — читай раздел [управление доступом](../fnox/team-access.md).
